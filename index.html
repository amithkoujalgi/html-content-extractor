Link Content Extractor API

About: Pass a link and get back the metadata about the link resource in a structured format. API reference: The API takes the URI of target and parses the result and sends the JSON structured response Sample response: {} Methods supported: [GET] You can make a HTTP GET request by passing the URI link in the request URL.
http://fathomless-gorge-9230.herokuapp.com/api/query/[your-link-here]
[POST] You can even make a HTTP POST request by posting data with parameter named 'link'.
http://fathomless-gorge-9230.herokuapp.com/api/extract
If you are using Ajax, you can use only GET method and POST cannot be used since cross-domain Ajax POST is not allowed as per Same origin policy Here's how to use Ajax GET :
var urlString = "http://fathomless-gorge-9230.herokuapp.com/api/query/"; $.getJSON(urlString + encodeURIComponent(givenURL), function(data, status){ if(status==="success"){ alert(JSON.stringify(data)); } else{ alert("failure"); } });
Test: With Curl Get:
curl http://fathomless-gorge-9230/api/query/[your-uri-encoded-link-here]
Post:
curl --data "{"link": [your-link-here]}" http://fathomless-gorge-9230/api/extract
